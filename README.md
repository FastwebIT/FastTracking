# 📈 Python Price Tracker Bot

Questo è un bot automatico scritto in Python che monitora i prezzi dei siti di e-commerce e invia una notifica in tempo reale su Telegram quando il prezzo scende sotto una soglia prestabilita.

## ✨ Funzionalità
- **Scraping in tempo reale:** Analizza le pagine web bypassando i blocchi base usando gli opportuni User-Agent.
- **Notifiche push istantanee:** Integrazione diretta con l'API di Telegram.
- **Leggero ed efficiente:** Progettato per essere eseguito 24/7 su un Raspberry Pi o su un server VPS gratuito.

## 🛠️ Come installarlo
1. Clona la repository: `git clone https://github.com/FastwebIT/FastTracking`
2. Installa i requisiti: `pip install -r requirements.txt`
3. Configura il file con i tuoi dati (Token Telegram e URL).
4. Avvia: `python fasttracking.py`
